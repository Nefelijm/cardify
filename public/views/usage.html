¿Como se debe linkear?

Para compilar el CSS Para compilar el JS

¿Como se debe llamar en el js?

$(document).ready(function(){ $('.example-container').cardify();});

Estructura

A traves de una funcion cardify llamaremos a nuestro plugin

//example-container es el selector del contenedor $('.example-container').cardify();

Al llamar la funcion cardify automaticamente empieza a ejecutar el siguiente codigo

(function($) { $.fn.cardify = function() { // Agregando estilos al contenedor $(this).addClass('cardifyContainer'); // Iterar entre imágenes $('img').each((index, image) => { // Creando elementos $(image).wrap(' '); $(image).parent().append($(' ').text($(image).attr('alt'))); // Agregando estilos a la imagen $(image).addClass('cardifyImage'); // Agregando estilos al figure $(image).parent().addClass('cardifyFigure'); // Agregando estilos al Figcaption $(image).next().addClass('cardifyFigcaption'); }); }; })(jQuery);
