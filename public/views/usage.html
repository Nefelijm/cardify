¿Que Archivos se deben usar?

Los Archivos que hay que tener en cuentas son: 
jquery.cardify.css
jquery.cardify.js

¿Como se debe linkear?

Para compilar en el CSS 
rel="stylesheet" href="css/jquery.cardify.css"
Para compilar el JS 
  src="js/jquery.cardify.js""

¿Como se debe llamar en el js?

$(document).ready(function(){ $('.example-container').cardify();});

Estructura

A traves de una funcion cardify llamaremos a nuestro plugin
example-container es el selector del contenedor 

$('.example-container').cardify();

Al llamar la funcion cardify automaticamente empieza a ejecutar el siguiente codigo 


							(function($){ $.fn.cardify = function()
								 { // Agregando estilos al contenedor $(this).addClass('cardifyContainer'); 
								  // Iterar entre imágenes $('img').each((index, image) => { 
								  // Creando elementos $(image).wrap('');
								  	$(image).parent().append($('').text($(image).attr('alt'))); 
								  // Agregando estilos a la imagen $(image).addClass('cardifyImage'); 
								  // Agregando estilos al figure $(image).parent().addClass('cardifyFigure');								
						  	  // Agregando estilos al Figcaption $(image).next().addClass('cardifyFigcaption');	
						    }); 
					    }; 
				    })(jQuery);
